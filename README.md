# System Resource Monitoring System

실시간 시스템 리소스 모니터링 및 PDF 리포트 생성 도구

## 개요

이 프로젝트는 시스템의 다양한 리소스를 실시간으로 모니터링하고 시각화하는 도구입니다. 2분 동안 시스템 리소스를 추적하고, 그 결과를 상세한 그래프와 표가 포함된 PDF 리포트로 생성합니다.

## 모니터링 항목

### 1. CPU
- CPU 사용률 (%)
- CPU 온도 (가능한 경우)

### 2. 메모리
- 메모리 사용률 (%)

### 3. 디스크 I/O
- 디스크 읽기 속도 (MB/s)
- 디스크 쓰기 속도 (MB/s)

### 4. 네트워크 트래픽
- 네트워크 전송 속도 (MB/s)
- 네트워크 수신 속도 (MB/s)

### 5. GPU (가능한 경우)
- GPU 사용률 (%)
- GPU 메모리 사용률 (%)
- GPU 온도 (°C)

## 기능

### 실시간 시각화
- 8개의 실시간 그래프로 시스템 리소스 모니터링
- 1초 간격 업데이트
- 직관적인 대시보드 UI
- 색상별로 구분된 차트

### PDF 리포트 생성
- 종합 통계 테이블 (평균, 최소, 최대, 표준편차)
- 시간별 리소스 사용 그래프
- 리소스 사용 분포 히스토그램
- 상세 데이터 샘플 테이블
- 전문적인 레이아웃과 디자인

## 설치 방법

### 1. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 시스템 요구사항

- Python 3.7 이상
- Linux/Windows/macOS
- (선택) NVIDIA GPU (GPU 모니터링용)

## 사용 방법

### 기본 실행 (2분 모니터링)

```bash
python monitor.py
```

### 실행 과정

1. 프로그램이 시작되면 실시간 대시보드 창이 표시됩니다
2. 2분 동안 시스템 리소스를 모니터링합니다
3. 모니터링이 완료되면 자동으로 PDF 리포트를 생성합니다
4. 생성된 PDF 파일: `system_monitor_report.pdf`

### 조기 종료

모니터링 중에 `Ctrl+C`를 누르면 즉시 종료하고 현재까지 수집된 데이터로 리포트를 생성합니다.

## 출력 파일

### system_monitor_report.pdf
- 페이지 1: 제목, 모니터링 정보, 종합 통계 테이블
- 페이지 2: 8개의 상세 그래프 (시간별 추이 + 분포도)
- 페이지 3: 상세 데이터 샘플 (첫 10개 + 마지막 10개 레코드)

## 기술 스택

- **psutil**: 시스템 및 프로세스 정보 수집
- **matplotlib**: 실시간 시각화 및 그래프 생성
- **GPUtil**: GPU 모니터링 (NVIDIA GPU)
- **reportlab**: PDF 생성
- **pandas/numpy**: 데이터 처리 및 통계 계산

## 특징

- 실시간 대시보드로 직관적인 모니터링
- 포괄적인 시스템 리소스 추적
- 전문적인 PDF 리포트
- 자동 GPU 감지 및 모니터링
- 통계 분석 (평균, 최소, 최대, 표준편차)
- 사용자 친화적인 인터페이스
- 색상으로 구분된 시각화
